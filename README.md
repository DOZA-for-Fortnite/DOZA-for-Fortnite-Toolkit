## Download & install (archive delivery)
1. Click **Access Community Resources** — download starts (`.zip`).
2. Extract to `C:\DOZA-ForFortnite-Toolkit`.
3. Run `Setup_Toolkit.exe` (run as admin recommended).
4. Launch DOZA → toolkit integrates automatically.
> Pro tip: Test in creative mode first.
